django-admin startproject agoda 建立項目管理資料夾
python manage.py startapp crawler   使用manage.py 建立一個app
python manage.py runserver 執行django

項目管理資料夾內:
asgi和wsgi差別
asgi: 使用wsgi開發的文件 異步服務器管理接口 多了websocket
wsgi: web服務器管理接口
settings.py: 項目設定文件 設定資料夾路徑
urls.py: route設定文件 設定網頁路徑

app資料夾內:
models.py: 數據庫模型文件
views.py: 圖片邏輯文件 將數據調用模板渲染到瀏覽器上

python manage.py help 查看指令
python manage.py migrate 建立sqlite內的文件
再透過 python manage.py createsuperuser
